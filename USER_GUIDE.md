# 畑管理アプリ — 使い方ガイド

## はじめに

このアプリは、家庭菜園の区画ごとに野菜を登録し、作業タスクや収穫を記録するための管理アプリです。  
スマートフォンのブラウザからアクセスして使います（インストール不要）。

---

## 起動とスプラッシュ画面

アプリを開くと、季節のイラストと農園名、名言が表示されます。  
画面をタップすると本編に進みます。

---

## グリッド画面（メイン画面）

アプリのメイン画面です。畑を上から見た区画図をイメージしています。

### 区画を登録する（管理者のみ）

1. グリッドの空きセルを **横にドラッグ** して範囲を選択する
2. 「区画を登録」ダイアログが開く
3. 野菜を選び、作業開始日を入力して **「登録」** を押す

> 複数列にまたがる区画（例：3列分のトウモロコシ）も登録できます。  
> 縦方向の連結には現在対応していません。

### 区画をタップして管理画面へ

登録済みの区画（色のついたセル）をタップすると、その野菜の管理画面が開きます。

### 管理中の作物リスト

グリッドの下に、現在管理中の全区画が一覧表示されます。  
進捗率・ステータス（生育中 / 収穫中 / 準備中）が確認できます。

---

## 区画管理画面

選択した区画の詳細を管理します。4つのタブがあります。

### 🌱 基礎知識タブ

野菜マスタでAI生成した基礎知識（サイズ・収穫目安・適正pH・育成時期など）が表示されます。  
参考URLにYouTubeのURLが登録されている場合、動画がタブ直下に表示されます。

### 🗺 ロードマップタブ

AIが生成した作業タスクが時系列で並びます。

**タスクをチェックする**
- チェックボックスをタップ → 実施日を選択して「記録」
- 同じタスクを複数日に記録したい場合は「＋追加」をタップ
- 記録済みの日付チップをタップすると日付変更・削除ができる

**進捗バー** はチェック済みタスク数をもとに自動計算されます。

### 🧺 収量タブ

収穫した量を記録します。

- 数量・単位（kg / 個 / 束など）・メモを入力して追加
- 累計収穫量が自動集計されます

### 📋 作業ログタブ

チェックしたタスクの記録が日付順に一覧表示されます（自動生成）。

**管理を完了する（管理者のみ）**  
作業ログタブの最下部にある「この野菜の管理を完了」ボタンで区画をアーカイブします。  
完了するとグリッドから区画が消え、「過去データ」に保存されます。

---

## 野菜マスタ

グリッド画面上部の **「野菜マスタ」** ボタンから開きます。

### 野菜の追加（管理者のみ）

「＋新しい野菜を追加」から野菜名・品種・絵文字・育成方法・参考URLを入力します。  
次へ進むとマスタ詳細が開き、追加の情報編集やAI生成ができます。

### ロードマップと基礎知識をAI生成する（管理者のみ）

野菜マスタで野菜を選択すると、「再生成」ボタンが表示されます。  
参考URLと育成方法をもとに、AIがロードマップと基礎知識を自動作成します。

> **注意**: 再生成するとロードマップのタスクが新規IDで作り直されます。  
> 区画に登録中の野菜を再生成すると、その区画のチェック記録がリセットされます。

### 保存する

ツールバー右端の **「保存」** ボタンで変更を保存します。  
変更後は必ず保存してください。

---

## 過去データ（アーカイブ）

完了した区画の記録を閲覧できます。

- 野菜ごとにカード表示
- スナップショットカレンダーで過去の畑の状態を確認
- 作業ログ・収穫記録・基礎知識を閲覧（編集不可）

---

## 設定

グリッド画面上部の **「設定」** ボタンから開きます。

| 設定項目 | 内容 |
|---|---|
| 農園名 | スプラッシュとグリッド画面に表示する名前 |
| グリッドサイズ | 列数（2〜20）・行数（2〜20）を変更 |
| 管理者ログイン | パスワード入力で管理者モードに切り替え |
| 全データリセット | ⚠️ 全データを削除（取り消し不可） |

---

## 管理者と閲覧者の違い

| 操作 | 閲覧者 | 管理者 |
|---|---|---|
| タスクのチェック・記録 | ✓ | ✓ |
| 収穫の追加 | ✓ | ✓ |
| 全画面の閲覧 | ✓ | ✓ |
| 区画の登録・削除 | — | ✓ |
| 野菜マスタの編集 | — | ✓ |
| AI生成 | — | ✓ |
| 設定変更 | — | ✓ |
| 管理の完了（アーカイブ） | — | ✓ |

---

## データの保存について

- 操作のたびに自動保存されます（手動保存は野菜マスタのみ）
- データはクラウドに同期されるため、複数の端末で共有できます
- アプリを閉じてもデータは保持されます

### 取り消し（Undo）

グリッド画面のツールバー左端に「取り消し」ボタンがあります（管理者のみ）。  
区画の登録・削除など直前の操作を取り消せます。

---

## よくある質問

**Q. 野菜マスタに野菜を追加したのにグリッドの選択肢に出ない**  
A. 区画登録ダイアログを一度閉じて再度開いてください。

**Q. ロードマップのタスクを自分で追加・編集したい**  
A. 野菜マスタのロードマップ内の「＋ タスクを追加」から追加できます。タスク名・日数・メモも編集可能です。

**Q. 参考動画が基礎知識タブに表示されない**  
A. 野菜マスタの「参考URL」欄にYouTubeのURLが登録されているか確認してください。

**Q. 複数人で使いたい**  
A. URLを共有するだけで閲覧できます。管理者パスワードを共有すれば編集も可能です。
